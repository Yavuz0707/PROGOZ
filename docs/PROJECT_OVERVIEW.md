# Project Overview

PROGOZ, klasik kayit yapan kamera sistemlerine proaktif analiz katmani ekleyen bir web uygulamasidir. Amaci kavga, fiziksel saldiri, yogun ve kaotik hareket gibi durumlari otomatik tespit ederek operator panelinde anlik uyari ve olay kaydi olusturmaktir.

Kullanici rolleri `admin` ve `operator` olarak modellenmistir. Admin kamera ve kullanici operasyonlarini yonetebilir; operator analiz ve olay ekranlarini kullanir.

Ana ozellikler: canli kamera analizi, video upload analizi, olay kaydi, snapshot, islenmis video, CUDA durum bilgisi, WebSocket bildirimleri ve modern dark dashboard.

Demo senaryosu: login, dashboard, video upload, analiz progress, islenmis video, olay detayi, webcam/RTSP canli analiz.
